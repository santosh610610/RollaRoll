How to Use RollaRoll

What is RollaRoll?

RollaRoll is a simple, colorful attendance tracking application designed specifically for music classes. It helps instructors keep track of which students attend which classes, with options to mark students as present, absent, late, or excused.

Getting Started

  1. Add Students - First, add all your students to the system
  2. Create Classes - Create different music classes (Piano, Guitar, Violin, etc.)
  3. Assign Students to Classes - For each class, assign the specific students who attend
  4. Take Attendance - Record attendance for each class session
  5. View Records - Check past attendance records anytime

Step-by-Step Guide

1. Managing Students

Click on the Students menu item to view all students. Use the "Add New Student" button to add students to the system. Each student needs only a name to be added.

2. Managing Classes

Click on the Classes menu item to view and manage your music classes. Use the "Add New Class" button to create a new class (e.g., "Piano Beginner", "Guitar Advanced").

3. Assigning Students to Classes

From the Classes page, click the Manage Students button next to a class. This allows you to select which students attend this specific class. Students can be in multiple classes.

4. Taking Attendance

From the Classes page, click the Take Attendance button for a class. Select the date and mark each student as:

  • Present - Student attended class
  • Absent - Student did not attend
  • Late - Student arrived late
  • Excused - Student had an excused absence

5. Viewing Attendance Records

Click on the Attendance menu item to see all attendance records. You can view details of any past attendance session by clicking "View Details".

Tips for Using RollaRoll

  • Add all students before creating classes for easier assignment
  • Take attendance regularly to maintain accurate records
  • Use the "Manage Students" feature whenever class rosters change
  • Check past attendance records to identify patterns

RollaRoll is built using the ZenCoder AI tool
